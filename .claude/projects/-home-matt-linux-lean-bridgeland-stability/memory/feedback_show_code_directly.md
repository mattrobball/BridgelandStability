---
name: Show code directly
description: When user asks to see code, display the actual code inline — don't just point at file paths or line numbers
type: feedback
---

When the user asks to see code, show the actual code content inline in the response. Don't just reference file paths or line numbers — that's useless to them.

**Why:** The user wants to read the code directly, not navigate to it themselves.

**How to apply:** Always paste the relevant code in the response. File locations alone are not helpful.
